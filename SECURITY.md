# Security Policy

If you find a security vulnerability, please report it privately using GitHub's **Private Vulnerability Reporting**.

1. Go to the **Security** tab of the repository.
2. Select **Vulnerability reporting** from the sidebar.
3. Click **Report a vulnerability**.

Alternatively, you can email [contact@tristanbudd.com](mailto:contact@tristanbudd.com). I'll acknowledge your report within 48 hours.
